<!--...-- >

Se usa para insertar comentarios en el código. Estos no se muestran en los navegadores. Es útil para explicar el código y poder retomar esto al hacer ediciones posteriores.

Ejemplo de uso

<!--This is a comment. Comments are not displayed in the browser-->

<!DOCTYPE>

La declaración<!DOCTYPE>define el tipo de documento que es. Todos los documentos HTML deben de empezar con esta declaración. Esta es información para que el navegador sepa que tipo de documento es.

Ejemplo de uso

<!DOCTYPE html>

<a>

Define un hipervínculo o liga y se usa para ir de una página web a otra. El elemento más importante de la sintaxis de esta etiqueta es<href>que indica el destino.
Por default los links aparecen en los navegadores con el siguiente formato:

  • Subrayado en azul para no visitado
  • Subrayado en morado para visitado
  • Subrayado en rojo para activo

Ejemplo de uso

<a href="https://www.w3schools.com">Visit W3Schools.com!</a>

<cite>

La etiqueta<cite> es usada para definir el título de una obra (libro, poema, canción, pintura,etc) El texto aparece formateado enitalicas.

Ejemplo de uso

<p><cite>The Scream</cite> by Edward Munch. Painted in 1893.</p>

<embed>

La etiqueta<embed> un contenedor para un recurso externo como una página web, una imagen, un reproductor de audio/video o un plug-in.

Ejemplo de uso

<embed type="image/jpg" src="example.jpg" width="300" height="200">

<h1> - <h6>

Estas etiquetas son usadas para definir encabezados, siendo h1 el encabezado o título con mayor jerarquía y tamaño hasta llegar al h6 siendo el que tiene menor tamaño.

Ejemplo de uso

<h1>This is heading 1</h1>

<h2>This is heading 2</h2>

<h3>This is heading 3</h3>

<h4>This is heading 4</h4>

<h5>This is heading 5</h5>

<h6>This is heading 6</h6>

<link>

La etiqueta<link>define la relación entre el documento actual y un recurso externo. Es usada frecuentemente para ligar una hoja de estilos CSS o añadir un favicon al sitio web. Es una etiqueta que solo contiene atributos.

Ejemplo de uso

<head>

<link rel="stylesheet" href="styles.css">

</head>

<nav>

La etiqueta <nav> define un set de ligas de navegación. No todos las ligas de navegación deben de ser incluidas en esta etiqueta solamente los bloques principales.

Ejemplo de uso

<nav>

<a href="/html/">HTML</a> |

</nav>

<pre>

La etiqueta<pre>define un texto con un formato predefinido, es decir que tal cual como aparece en el documento HTML es como se va a mostrar.

Ejemplo de uso

<pre>

Lorem ipsumdolor

sit amet

consectetur

</pre>

<section>

Define una sección dentro de un documento.

Ejemplo de uso

<section>

<h2>Lorem ipsum</h2>

<p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Aut beatae ex quis voluptatum, alias, cupiditate vero temporibus tenetur modi perferendis iure accusantium quisquam mollitia labore, vitae cum debitis error hic?</p>

</section>
